# Habit Tracker PWA

## Overview
A single-page Progressive Web App for tracking daily habits. Fully offline-capable, data stored in `localStorage`.

## Project Structure
```
habit_tracking/
├── index.html       # Full app (HTML + CSS + JS inline)
├── sw.js            # Service Worker for offline caching
├── manifest.json    # PWA manifest for installability
├── icon-192.png     # App icon (192x192)
├── icon-512.png     # App icon (512x512)
└── README.md        # Project documentation
```

## Key Architecture Decisions
- **Single file app**: All HTML, CSS, and JS live in `index.html` — no build tools or dependencies
- **localStorage**: Used for all data persistence — habits array with completion dates
- **Service Worker**: Cache-first strategy, caches all assets on install for full offline support
- **PWA manifest**: Enables "Add to Home Screen" on mobile browsers

## Data Format
```json
{
  "habits": [
    {
      "name": "Exercise 30 min",
      "completed": ["2026-02-17", "2026-02-18"]
    }
  ]
}
```

## Deployment
- Hosted via GitHub Pages at `https://abdokaseb.github.io/habit-tracker/`
- To update: push changes to `main` branch

## Development
- No build step needed — edit `index.html` directly
- Test locally: `python -m http.server 8080` from this directory
- PWA install requires HTTPS (GitHub Pages provides this automatically)
